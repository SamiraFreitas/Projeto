## 💻 Como baixar o projeto 

```bash
#Clonando o repositorio 
$ git clone https://github.com/SamiraFreitas/Projeto
#Entrar na pasta
$ cd Projeto
# Instalar as dependências
$ yarn install
#Rode o projeto 
$ yarn dev
```
